> HDLS is a personal research collective studying local AI, mesh networking,
> knowledge systems, and embedded firmware. Everything runs on hardware I own.
> No cloud dependencies, no API keys, no telemetry. The work is organized
> through a signal chain of agents that hunt, read, ingest, and design.

## Signal Chain

The HDLS pipeline runs as a signal chain of specialized agents. Each stage
shapes the signal before passing it to the next:

```
  dr3dg3        pr0b3        g0blin
    |             |             |
    v             v             v
  hunt  ------>  read  ------>  ingest  ------>  vault
    |             |             |                |
    |             |             |                |
    +-------------+-------------+----------------+
                          |
                          v
                       0r4cl3
                    (design keeper)
```

| Agent | Role |
|-------|------|
| dr3dg3 | Source hunter, crawler |
| pr0b3 | Deep-work researcher |
| g0blin | Vault gatekeeper, triage |
| 0r4cl3 | Design keeper, brand steward |
| hyph4 | Coordinator, Discord presence |
| s3ntry | Security auditor |
| 3tch | Code implementation |
| r3nd3r | Visual production |
| spl1c3 | Content assembly |
